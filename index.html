O que é mascara de sub-rede ?

A máscara de sub-rede é um número de 32 bits que é usado para dividir um endereço IP em duas partes: o endereço de rede e o endereço de host. A máscara de sub-rede é usada pelos roteadores para determinar para qual rede um pacote de dados deve ser encaminhado.

Os endereços IP são divididos em quatro octetos, cada um com 8 bits. A máscara de sub-rede também é dividida em quatro octetos, com cada octeto representando uma parte do endereço IP. Os bits da máscara de sub-rede são definidos como 1 ou 0. Os bits 1 representam o endereço de rede, enquanto os bits 0 representam o endereço de host.

Por exemplo, um endereço IP de 192.168.1.100 pode ser dividido em duas partes usando a máscara de sub-rede 255.255.255.0. Os primeiros três octetos do endereço IP (192.168.1) são o endereço de rede, enquanto o último octeto (100) é o endereço de host.

Os roteadores usam a máscara de sub-rede para comparar o endereço de destino de um pacote de dados com a máscara de sub-rede da rede. Se os bits correspondentes da máscara de sub-rede forem iguais, o roteador determina que o pacote de dados deve ser encaminhado para a rede especificada pelo endereço de rede.

A máscara de sub-rede é uma parte importante da arquitetura de redes TCP/IP. Ela permite que as redes sejam divididas em sub-redes menores, o que pode melhorar a eficiência e a segurança da rede.

Aqui estão alguns dos benefícios de usar uma máscara de sub-rede:

  • Melhora a eficiência da rede: Ao dividir uma rede em sub-redes menores, os roteadores podem encaminhar pacotes de dados mais rapidamente e com mais eficiência.
  • Melhora a segurança da rede: Ao dividir uma rede em sub-redes menores, os administradores de rede podem restringir o acesso a recursos específicos.
  • Permite a realocação de endereços IP: Ao usar uma máscara de sub-rede, os administradores de rede podem realocar endereços IP sem interromper o tráfego da rede.

    • A máscara de sub-rede é um conceito importante para entender se você está trabalhando com redes TCP/IP.

    A sub-rede de uma rede IP é separar uma grande rede em redes múltiplas menores para fins de reorganização e segurança. Todos os nós (hosts) em uma subrede consulte todos os pacotes transmitidos por qualquer nó em uma rede. O desempenho de uma rede é afetado negativamente a carga pesada do tráfego devido às colisões e às retransmissãos.

    A aplicação de uma máscara de sub-rede a um endereço IP separa o endereço de rede do endereço do host. Os bits de rede são representados pela 1 ‘ s na máscara e os bits de host são representados por 0 ‘ s. A execução de uma operação lógica e de bit de bits no endereço IP com a máscara de sub-rede produz o endereço de rede. Por exemplo, a aplicação da máscara de sub-rede da classe C ao nosso endereço IP 189.3.128.12 produz o seguinte endereço de rede:

  • IP: 1101 1000 . 0000 0011 . 1000 0000 . 0000 1100 (189.003.128.012)
  • Mask: 1111 1111 . 1111 1111 . 1111 1111 . 0000 0000 (255.255.255.000)
